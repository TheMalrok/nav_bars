# 🚀 Szkielet Aplikacji Flutter - Projekt Szkoleniowy

Witaj w projekcie szkoleniowym! Ten projekt jest zaledwie prostym szkieletem demonstracyjnym i stanowi punkt wyjścia do Twojej pracy oraz nauki. Obecnie zawiera makietę interfejsu opartą na dolnym pasku nawigacyjnym (Bottom Navigation Bar) i zarys wyglądu kilku ekranów.

Twoim zadaniem jest "ożywienie" aplikacji poprzez wdrożenie zaawansowanych funkcjonalności.

## 🎯 Główne cele dla uczniów

Docelowo w ramach ćwiczeń musisz uzupełnić ten szkielet o następujące zaawansowane funkcje:

1. **🎨 Motywy (Theme i Dark Mode)**
   - Wdrożenie obsługi motywów.
   - Możliwość dynamicznego włączania/wyłączania motywu ciemnego z poziomu ekranu ustawień.

2. **🌍 Wielojęzyczność i tłumaczenia (i18n/l10n)**
   - Implementacja wsparcia dla co najmniej dwóch języków (np. polski i angielski).
   - Pozwalanie użytkownikowi na zmianę języka aplikacji w locie.

3. **⚙️ Zaawansowane funkcje systemu**
   - Wdrożenie bezpiecznego przetrzymywania stanu użytkownika (np. czy jest zalogowany).
   - Zastąpienie makiet (MockUI) rzeczywistym stanem z systemu.
   - Korzystanie z funkcji platformy np. systemowych powiadomień lub pamięci lokalnej.

4. **🔄 Rozwój Logiki Ekranów**
   - Rozwinięcie `ProfileScreen` o możliwość edycji profilu.
   - Skonfigurowanie `SettingsScreen` aby reagował i zapisywał wybrane przez użytkownika opcje.

## 🛠 Uruchamianie

1. Pobierz repozytorium na swój dysk.
2. Zaktualizuj i pobierz paczki poleceniem:
   ```bash
   flutter pub get
   ```
3. Uruchom aplikację na wybranym emulatorze lub urządzeniu:
   ```bash
   flutter run
   ```

Powodzenia i owocnej nauki!
